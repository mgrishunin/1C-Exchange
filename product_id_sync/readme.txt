Маханизм ежедневной синхронизации кодов товара 1с и сайта.

1. Регламентное задание "Выгрузка товаров" еженощно выгружает все товары из интеерсующих нас товаров в папку c:/obmen/ сервера
2. Т.к. товаров много,  задаем в 1С "использовать порционную  выгрузку" кол-во товаров в порции 500-700
3. На сервер устанавыливаем WinSCP, настраиваем FTP доступ к папке obmen web сервера
4. В планировщике заданий windows создаем задание запускать "C:\Program Files (x86)\WinSCP\WinSCP.exe" с параметрами  /console  /script="C:\scp_sync.txt"
5. В результате выгруженные 1С файлы оказываются на серваере в папке obmen
6. В КРОН задаем ваыполнять каждые 5 минут скрипт xml_sync.php
7. Данный скрипт берет первый файл в папке, добавляет 1С коды из него в СУБД и удаляет этот файл
8. Так происхожит пока все файлы в папке не будут обработаны и удалены
